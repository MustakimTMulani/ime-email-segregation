# IME Shipping Email Segregation System

## Overview

Automated shipping email classification and data extraction system.

## Features

- Email Classification
- Tonnage Extraction
- Cargo VC Extraction
- Cargo TC Extraction
- Multi-Record JSON Output
- Duplicate Detection
- Vessel-Cargo Matching
- Dashboard & Analytics

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- Bootstrap
- Regex + Heuristic NLP

## API Endpoint

POST /api/extract

Returns structured JSON from shipping emails.
